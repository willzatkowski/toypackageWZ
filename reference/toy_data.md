# A toy dataset for demonstrating the functions in toypackage

A toy dataset for demonstrating the functions in toypackage

## Usage

``` r
toy_data
```

## Format

### `toy_data`

A data frame with 5 rows and 4 columns:

- subject_id:

  An integer denoting the subject id

- reaction_time:

  Reaction time in seconds

- test_score:

  Test score of each subject

- recalled_words:

  Words recalled by each subject
